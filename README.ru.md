# Train Schedule Bot 🚆

Telegram-бот, который показывает ближайшие поезда между Дзержинском и Минском и рассчитывает, во сколько нужно выйти из дома, чтобы успеть на поезд — на основе твоей текущей геопозиции.

## Возможности

- 📅 Показывает 3 ближайших отправления поезда в выбранном направлении
- 📍 Принимает GPS-геопозицию через Telegram
- 🚶 Автоматически рассчитывает время пешком до станции (через OpenRouteService API)
- ⏰ Сообщает точное время выхода и номер поезда, на который успеваешь
- 💾 Сохраняет ID пользователей локально для базовой статистики

## Стек технологий

- **Python 3**
- [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI) — фреймворк для Telegram-ботов
- [OpenRouteService API](https://openrouteservice.org/) — расчёт пешего маршрута и времени
- `python-dotenv` — работа с переменными окружения

## Структура проекта

```
├── main.py           # Точка входа, запуск бота, хранение пользователей
├── config.py         # Переменные окружения и расписание
├── handlers.py        # Обработчики сообщений / логика бота
├── keyboards.py       # Клавиатуры Telegram
├── storage.py          # Хранение геопозиций пользователей в памяти
├── utils.py            # Поиск ближайших поездов, расчёт времени
├── requirements.txt
└── users.json.example  # Пример структуры файла с ID пользователей
```

## Как это работает

1. Пользователь пишет `/start` и получает меню с тремя опциями
2. **«3 ближайших поезда»** — показывает 3 ближайших отправления в выбранном направлении
3. **«Отправить GPS»** — пользователь делится геопозицией
4. **«Расчет»** — бот рассчитывает время пешком до станции через OpenRouteService, затем вычисляет, во сколько нужно выйти и на какой поезд получится успеть

## Запуск локально

1. Склонируй репозиторий:
```bash
git clone https://github.com/ahtemka22/train-schedule-bot.git
cd train-schedule-bot
```

2. Установи зависимости:
```bash
pip install -r requirements.txt
```

3. Создай файл `.env` в корне проекта со своими данными:
```
BOT_TOKEN=твой_токен_телеграм_бота
ORS_API_KEY=твой_ключ_openrouteservice
```

- Токен бота получи у [@BotFather](https://t.me/BotFather) в Telegram
- Бесплатный API-ключ получи на [openrouteservice.org](https://openrouteservice.org/dev/#/signup)

4. Запусти бота:
```bash
python main.py
```

## Примечания

- Расписание поездов сейчас захардкожено в `config.py` для направления Дзержинск–Минск. Это можно расширить, подключив живые данные о расписании или добавив другие направления.
- `users.json` исключён из репозитория (см. `.gitignore`), так как хранит реальные Telegram ID пользователей; `users.json.example` показывает ожидаемый формат.
